Single purpose plugin to disable right-click on selected pages of your WordPress website.

== Description ==

= Single purpose plugin with one job: Disable right-click on selected pages of your website =

Ideal to prevent users from downloading the images from your site.

Disclaimer: Images can still be downloaded if the user knows how, but at least it won't be as simple as right-click and download.

== Frequently Asked Questions ==

= Will this plugin prevent people from downloading images from my website? =

It will only disable the right-click function, so they won't be able to right-click to download the images. However, they will be able to use other methods to download the images.

= Can I use this plugin to disable right-click on all the pages of my website at once? =

No, this plugin is designed to disable right-click on certain pages, so you would have to edit every page to disable right-click on all your pages.

== Installation ==

1. Upload the `8 Disable Right Click` plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You will see a "Disable Right Click" window in the editing area of every page in your website.
4. Select "Yes" and save changes in every page you want to use it.
